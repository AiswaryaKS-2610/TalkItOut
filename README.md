# Automated Backend Deployment with Railway & MySQL

A simple backend application built with Node.js and Express, connected to a cloud-hosted MySQL database on Railway. The project demonstrates REST API development, cloud database integration, and production-style configuration without Docker.

---

## Features
- REST APIs using Node.js and Express
- MySQL database hosted on Railway
- Automatic table creation on startup
- Full CRUD operations (Create, Read, Update, Delete)
- Environment-based configuration
- Cloud deployment on Railway (no Docker)

---

## Tech Stack
- **Backend:** Node.js, Express
- **Database:** MySQL (Railway)
- **Cloud Platform:** Railway
- **Version Control:** Git, GitHub
